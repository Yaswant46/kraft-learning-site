# Kraft Calculator Academy

A clean learning site to master paper process engineering and feed those fundamentals into your Kraft Calculator app.

## Pages
- `/` — Landing page and roadmap
- `/modules.html` — 6 learning modules with code snippets
- `/graphs.html` — Mermaid flow + interactive D3 dependency graph
- `/videos.html` — Curated YouTube embeds (edit `/public/data/videos.json`)

## Deploy on Vercel
1. Push this folder to a GitHub repository.
2. Vercel → New Project → Import repository → Framework preset: "Other".
3. Vercel will detect `vercel.json` for routing static pages and (if needed) serverless functions.

## Customize
- Replace placeholder YouTube IDs in `public/data/videos.json`.
- Adjust styles in `public/styles.css`.
